**Last Updated:** February 8, 2026  
**Effective Date:** February 8, 2026

## Overview

Worth My Time is a Chrome browser extension that helps you make better purchasing decisions by converting product prices into working time equivalents. We are committed to protecting your privacy and being completely transparent about our data practices.

**Key Privacy Commitment:** Worth My Time does NOT collect, transmit, or share ANY user data.

---

## Data Collection

### What Data We Store Locally

Worth My Time stores the following information **exclusively on your device** using Chrome's local storage:

| Data Type | Example | Purpose | Storage Location |
|-----------|---------|---------|------------------|
| Monthly Salary (After Tax) | $5,000 | Calculate hourly rate | Chrome Local Storage |
| Work Hours per Day | 8 hours | Calculate hourly rate | Chrome Local Storage |
| Work Days per Month | 22 days | Calculate hourly rate | Chrome Local Storage |
| Extension Status | Enabled/Disabled | Remember user preference | Chrome Local Storage |

### What We DO NOT Collect

Worth My Time does **NOT** collect, access, transmit, or store:

- ❌ Personal identifying information (name, email, address, phone)
- ❌ Browsing history or website visits
- ❌ Shopping behavior or purchase history
- ❌ Credit card or payment information
- ❌ Location data or IP addresses
- ❌ Cookies or tracking identifiers
- ❌ Analytics or telemetry data
- ❌ Crash reports or diagnostics
- ❌ Device information
- ❌ Any data from websites you visit

## How We Use Data

### Local Processing Only

All calculations happen **entirely on your device**:

1. You enter your salary information
2. Extension calculates your hourly rate locally
3. Extension reads prices from webpages you visit
4. Extension displays time equivalents next to prices
5. **No data leaves your device at any point**

### No Remote Servers

Worth My Time:
- ✅ Does NOT connect to any external servers
- ✅ Does NOT make any network requests
- ✅ Does NOT transmit any data
- ✅ Works completely offline
- ✅ Has NO backend infrastructure

## Data Storage & Security

### Local Storage Only

All data is stored using Chrome's `chrome.storage.sync` API:

- **Storage Location:** Your local device only
- **Encryption:** Handled by Chrome's built-in encryption
- **Sync:** Only through Chrome's sync (if you enable it)
- **Control:** You have complete control over all data

### Chrome Sync (Optional)

If you enable Chrome sync across devices:
- Chrome itself may sync your extension settings
- This is entirely managed by Google Chrome
- Your data is encrypted during sync by Chrome
- Worth My Time does not control this sync process
- You can disable Chrome sync at any time

### Data Security Measures

- ✅ **Zero network access** - Extension never connects to internet
- ✅ **Minimal permissions** - Only requests essential permissions
- ✅ **Local processing** - All calculations done on your device
- ✅ **No external libraries** - No third-party code that could leak data
- ✅ **Manifest V3** - Uses latest Chrome security standards
- ✅ **Open source** - Code is publicly auditable

## Permissions Explained

Worth My Time requests these Chrome permissions:

### `storage`
- **Purpose:** Save your salary and work settings locally
- **Data Access:** Only settings you explicitly enter
- **Data Shared:** None - stays on your device
- **Required:** Yes - extension cannot function without saving settings

### `activeTab`
- **Purpose:** Read prices on the current webpage you're viewing
- **Data Access:** Only visible price text on current tab
- **Data Shared:** None - no data collected or transmitted
- **Required:** Yes - needed to detect and convert prices

### `host_permissions: ["<all_urls>"]`
- **Purpose:** Work on any shopping website you visit
- **Data Access:** Only reads visible price elements
- **Data Shared:** None - all processing is local
- **Required:** Yes - users shop on many different websites

### Why We Need These Permissions

Unlike many extensions, we use permissions **only** for core functionality:
- We DON'T use them to track you
- We DON'T use them to collect data
- We DON'T use them to show ads
- We DON'T use them to monetize your data
